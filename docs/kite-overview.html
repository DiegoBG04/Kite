Overview

What is Kite?

Kite is a personal investment intelligence platform built for investors who want more than price charts. It combines live market data, SEC filing analysis, and an AI assistant that understands your specific portfolio — not generic financial advice.

Unlike platforms that require institutional access or charge thousands per year, Kite is built to give individuals the same analytical depth that professional analysts use: revenue trend analysis, earnings comparisons, sector concentration risk, and AI-generated briefs — all anchored to your actual holdings.

Architecture

How It's Built

Frontend — React + Vite
Single-page app. Recharts for all data visualizations. Deployed on Vercel.
Backend — FastAPI (Python)
REST API with APScheduler for background jobs. Deployed on Railway.
Database — Supabase (PostgreSQL)
pgvector for semantic search on SEC filings. Quote and time-series cache tables reduce API calls to near-zero at scale.
AI — Claude (Anthropic)
RAG pipeline over SEC filings. Retrieves relevant chunks via pgvector, reasons with Claude, returns cited answers.
Market Data — TwelveData
Real-time quotes, historical time series, company stats. DB-first caching reduces API usage to ~26 credits/day at scale.
Financial Data — SEC EDGAR XBRL
Revenue, net income, FCF, cash, debt — pulled directly from SEC structured filings. No third-party financial data provider needed.
Background Jobs — APScheduler
Market quotes refreshed every 15 min during US market hours. Financial data re-ingested nightly at 2 AM UTC.
Embeddings — OpenAI text-embedding-3-small
SEC filing chunks embedded at ingestion time. Stored in pgvector for sub-100ms semantic retrieval.
